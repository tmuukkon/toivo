Taidot

Tälle sivulle olen koonnut IT-taitojani. Kaikkea en ole maininnut, vain tärkeimpiä asioita. En myöskään ole maininnut asioita, joista minulla on vain vähän tietoa tai olen käyttänyt niitä vain vähän, kuten JS, PHP ja C#. Pythonia osaan sen verran kuin OPS vaatii ja vähän päälle. :D

Windows Server
Microsoft

Olen asentanut, konfiguroinut ja käyttänyt Windows Serveriä moneen eri tarkoitukseen. Muun muassa toimialueen hallintaan (josta tällä sivulla lisää), web-palveluiden hostaamiseen ja tiedostojen säilyttämiseen (verkkolevyt, AD). Toimialueessa myöskin on toki sertifikaattien hallintaa.

Linux
Linus Torvalds

On kunnia suomalaisena käyttää Linus Torvaldsin kehittämään mahtavaan Linux-ytimeen (kerneliin) perustuvia käyttöjärjestelmiä! Käyttöjärjestelmänäni on Arch Linux. Olen myös käyttänyt muitakin Linux-jakeluita, kuten Fedoraa, Ubuntua (myös Server) ja Debiania sekä niihin perustuvia jakeluita kuten Linux Mintiä. Olen käyttänyt monia työpöytäympäristöjä kuten Gnomea ja KDE Plasmaa (jota käytän päivittäin).

AD DS
Windows Server, Microsoft

Olen asentanut ja käytän Windows Serverillä pyörivää Active Directory -hakemistoa/toimialuetta. Siihen sisältyy mm. laitehallinta ja tunnusten hallinta.

Käyttöjärjestelmät
Windows Client, macOS, iOS, Android

Monia eri laitteita eri käyttöjärjestelmillä on tullut käytettyä. iOS 6-18, Android 2-14, macOS 11 (Big Sur)-15 (Sequioa), Windows 3.11-11.

PowerShell
Microsoft

Olen käyttänyt PowerShelliä hallintaa helpottavien skriptien tekemiseen sekä toimintojen suorittamiseen. Skriptejä mm. turhien sisäänrakennettujen Windows 11 -ohjelmien poisto, Windowsin rekisteröinti- ja tukitietojen asetus sekä verkkolevyjen yhdistäminen.

Toimisto-ohjelmat
Microsoft Office, LibreOffice (ja OpenOffice), Apple iWork, Google Workspace, OnlyOffice

Excelin tai muiden taulukkolaskentaohjelmien funktioita en osaa.

Pilviympäristöt
Microsoft Azure, Microsoft 365, Google Workspace

Minulla on muutaman vuoden kokemusta pilviympäristön, kuten Microsoft 365:n ja Entra ID:n hallinnasta sekä turvallisesta ylläpidosta (mm. Microsoft Defender for Identityn avulla). Vielä ei ole tullut käytettyä Defender for Endpointia, sillä se ei sisälly Microsoft 365 E5 Developer -lisenssiin.

Ryhmäkäytäntö (GPO)/Intune
AD DS, Microsoft

Olen käyttänyt paikallisen eli on-premises AD-ympäristön ryhmäkäytäntöä eli GPO:ta sekä Microsoft Entra ID:n (ME-ID, ent. Azure AD) Intunea käytäntöjen lisäämiseksi tietokoneisiin ja mobiililaitteisiin sekä niiden hallinnointiin. Intunessa myös ohjelmistojakelu.

SCCM eli (Microsoft) System Center Configuration Manager (ConfigMgr)
AD DS, Microsoft

Olen asentanut ja konfiguroinut paikalliseen eli on-premises AD-ympäristöön SCCM-palvelun laitteiden hallintaan sekä ohjelmistojakelua varten (Software Center). Tulevaisuudessa sitten CMG:n konfigurointi, niin pääsee SCCM:ään käsiksi ulkoverkosta.

WordPress
Nettisivut

Olen käyttänyt WordPressiä sekä normaalien nettisivujen tekemiseen että ns. palveluiden tekemiseen, kuten sisäisen keskustelu/tiedostonjakoalustan sekä selaimen etusivuna toimivan työkalupakin tekemiseen.

HTML ja CSS
Nettisivut

Olen muutaman nettisivun, kuten tämän, tehnyt ihan vain HTML:llä ja CSS:llä. Tämä sivu on tehty valmiilla CSS-frameworkilla, Bootstrapilla.

Okta IAM (Identiteetin- ja pääsynhallinta, eng. Identity and Access Management)
Okta, Inc.

Olen käyttänyt Oktaa paikallisen AD-ympäristön että pilvessä olevan Entra ID-ympäristön tunnustenhallintaan.

GLPI ja ADManager
Paikallisessa eli on-premises AD-ympäristössäni käytössä olevat hyödylliset palvelut

GLPI on omaisuusrekisteri ja tukijärjestelmä. Siinä on inventaario-ominaisuus, jota käyttämällä voit kerätä tietoa laitteista, kuten tietokoneista, ja siihen kytketyistä lisälaitteista. Sinne saadaan myös lisättyä mobiililaitteita, liittymiä, palvelinräkkejä yms. ADManager on paikallisen AD-ympäristön hallintaa huomattavasti helpottava palvelu, jossa voi tehdä mm. käyttäjän luomiseen pohjia.

IDE:t (Integrated development environment) eli ohjelmointiympäristöt, koodausohjelmat
Microsoft Visual Studio, Microsoft Visual Studio Code (VS Code), JetBrains WebStorm

En oikeastaan kirjoita koodia, sillä HTML:ää eikä CSS:ää lasketa koodikieliksi, vaan ne ovat merkintäkieliä. Sivut teen pääsääntöisesti WebStormilla, mutta myös VS Codea tulee käytettyä. Visual Studiota olen käyttänyt Windows-ohjelmien tekemiseen. Tein Windowsin system trayssä eli ilmaisinalueessa elelevän valikko-ohjelman, josta voi avata tärkeitä AD-ympäristöön liittyviä asioita, kuten vaikkapa Software Centerin tai tietyn verkkolevyn.
